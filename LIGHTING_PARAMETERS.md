# פרמטרים השולטים על התאורה הכללית של הסצנה

## 1. DynamicLighting Component
זה הקומפוננטה הראשית ששולטת על התאורה הכללית של החדר.

### פרמטרים:

#### **ambientIntensity** (שורה 122)
- **מה זה**: תאורה סביבתית כללית - מאירה את כל האובייקטים באופן אחיד
- **טווח**: 0.02 (חשוך) ← → 0.3 (מואר)
- **השפעה**: הכי חשוב - זה משפיע על הבהירות הכללית של כל הסצנה
- **נוסחה**: `0.02 + (0.3 - 0.02) * (1 - darknessCurve)`

#### **directionalIntensity** (שורה 123)
- **מה זה**: תאורה כיוונית ראשית - כמו שמש
- **טווח**: 0.05 (חשוך) ← → 0.8 (מואר)
- **השפעה**: יוצר צללים וכיוון תאורה
- **נוסחה**: `0.05 + (0.8 - 0.05) * (1 - darknessCurve)`

#### **directional2Intensity** (שורה 124)
- **מה זה**: תאורה כיוונית משנית - תאורה משלימה
- **טווח**: 0.02 (חשוך) ← → 0.3 (מואר)
- **השפעה**: ממלאת צללים ומעניקה עומק

#### **spotIntensity** (שורה 125)
- **מה זה**: תאורת spot - אלומת אור ממוקדת
- **טווח**: 0.02 (חשוך) ← → 0.4 (מואר)
- **השפעה**: תאורה ממוקדת מלמעלה

#### **rimIntensity** (שורה 126)
- **מה זה**: תאורת rim - תאורה היקפית לעומק
- **טווח**: 0.02 (חשוך) ← → 0.2 (מואר)
- **השפעה**: יוצר גבולות וקונטור

## 2. CeilingLight Component
מנורת התקרה - אמורה להיות המקור העיקרי בחושך.

### פרמטרים:

#### **lightIntensity** (שורה 53)
- **מה זה**: עוצמת מנורת התקרה
- **טווח**: 0 (כבויה) ← → 8 (מלוא העוצמה)
- **השפעה**: מאירה את החדר בחושך
- **נוסחה**: `Math.pow(darkness, 1.2) * 8`

## הבעיה הנוכחית

התאורה הבסיסית (`DynamicLighting`) עדיין חזקה מדי גם בחושך:
- `ambientIntensity` מינימלי: 0.02 - עדיין מאיר
- `directionalIntensity` מינימלי: 0.05 - עדיין מאיר
- `spotIntensity` מינימלי: 0.02 - עדיין מאיר

**התוצאה**: מנורת התקרה לא בולטת כי התאורה הבסיסית עדיין חזקה.

## פתרון מוצע

להפחית עוד יותר את הערכים המינימליים:
- `ambientIntensity`: 0.01 ← → 0.3 (במקום 0.02)
- `directionalIntensity`: 0.01 ← → 0.8 (במקום 0.05)
- `directional2Intensity`: 0.01 ← → 0.3 (במקום 0.02)
- `spotIntensity`: 0.01 ← → 0.4 (במקום 0.02)
- `rimIntensity`: 0.01 ← → 0.2 (במקום 0.02)

או להעלות את עוצמת מנורת התקרה:
- `lightIntensity`: עד 12-15 במקום 8
