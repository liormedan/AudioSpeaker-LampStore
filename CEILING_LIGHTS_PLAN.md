# תיכנון מנורות תקרה - Dimmer Control

## מטרה
יצירת 1-2 מנורות תקרה חזקות שהמתג שולט עליהן, כך שהן מאירות את החדר בצורה דרמטית כאשר החדר חשוך.

## בעיות במימוש הנוכחי
1. **יותר מדי מנורות** - 9 מנורות יוצרות תאורה מפוזרת ולא דרמטית
2. **עוצמה נמוכה** - כל מנורה עם intensity 2.5 לא מספיק חזקה
3. **מיקום לא אופטימלי** - מפוזרות מדי, לא ממוקדות

## פתרון מוצע

### 1. מספר ומיקום מנורות
- **1-2 מנורות תקרה מרכזיות** במקום 9
- מיקום אופטימלי:
  - מנורה אחת: מעל מרכז החדר `[0, 9, 0]`
  - שתי מנורות: `[-4, 9, 0]` ו-`[4, 9, 0]` (משני צדי המרכז)

### 2. עוצמת תאורה
- **עוצמה גבוהה יותר**: 5-8 במקום 2.5
- **טווח תאורה רחב יותר**: distance 30-40
- **אפקט קונוס גדול יותר**: רדיוס 3-4, גובה 10-12

### 3. עיצוב מנורה
- **גוף גדול יותר**: רדיוס 0.4-0.5, גובה 0.3
- **נורה גדולה יותר**: רדיוס 0.2-0.25
- **אפקט זוהר חזק יותר**: emissiveIntensity גבוה יותר

### 4. לוגיקת שליטה
- **הפוך מהנוכחי**: 
  - darkness = 0 → מנורות כבויות (אור יום)
  - darkness = 1 → מנורות במלוא העוצמה (חושך)
- **עקומה חלקה**: easing function לעלייה חלקה

### 5. אפקטים ויזואליים
- **קונוס אור גדול** - נראה כמו אלומת אור מהתקרה
- **זוהר סביב המנורה** - אפקט halo
- **צללים** - castShadow מופעל

## מימוש טכני

### קומפוננטה: CeilingLights
```typescript
function CeilingLights({ darkness }: { darkness: number }) {
  // 1-2 מנורות מרכזיות
  // עוצמה: darkness * 6-8
  // distance: 35-40
  // אפקט קונוס גדול
}
```

### אינטגרציה עם DynamicLighting
- התאורה הבסיסית (ambient, directional) תישאר נמוכה בחושך
- מנורות התקרה יאירו את החדר בחושך
- בחושך מלא (darkness=1): מנורות חזקות + תאורה בסיסית מינימלית

## תוצאה צפויה
- **בחושך מלא**: מנורות תקרה חזקות מאירות את החדר, תאורה דרמטית
- **באור יום**: מנורות כבויות, תאורה טבעית
- **מעבר חלק**: בין מצבים שונים
